# 접근제어솔루션 RPM 설치 패키지 도입
- Linux 패키지 라이브러리 RPM을 활용하여 접근제어솔루션 서버 Installer, Patch 파일을 빌드 했습니다.
- .spec 파일에 Release 노트 및 패치 모듈 정보를 입력하여 Shell 상에서 언제든 패치 히스토리 파악이 가능 합니다.
- 현장 엔지니어가 간단한 명령어로 패치/롤백이 가능합니다.
